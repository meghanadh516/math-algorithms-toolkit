# Math Algorithms Toolkit

A comprehensive Python toolkit implementing core mathematical algorithms from scratch — covering algebra, calculus, matrix operations, graphs, and number theory.

## 🚀 Why This Project

This toolkit is designed to:
- Provide clean Python implementations of core math algorithms
- Help students, programmers, and competitive coders understand algorithm logic
- Serve as a learning resource for data structures and mathematics

## 📦 Features

✔️ Algebra: polynomial functions, equation solvers  
✔️ Calculus: numerical differentiation & integration  
✔️ Matrix: determinants, inverses, multiplications  
✔️ Graphs: BFS, DFS, shortest paths  
✔️ Number Theory: GCD, prime tests, modular arithmetic
